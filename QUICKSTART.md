## API Usage Examples

### cURL

```bash
curl -X POST http://localhost:8000/api/audit \
  -H "Content-Type: application/json" \
  -d '{"code": "import sqlite3\n\ndef query_user(user_id):\n    conn = sqlite3.connect(\"users.db\")\n    cursor = conn.cursor()\n    cursor.execute(f\"SELECT * FROM users WHERE id = {user_id}\")\n    return cursor.fetchone()"}'
```

### Python

```python
import requests

response = requests.post(
    "http://localhost:8000/api/audit",
    json={"code": "your_code_here"}
)

result = response.json()
print(f"Status: {result['final_status']}")
print(f"Iterations: {result['total_iterations']}")
```

### JavaScript

```javascript
const response = await fetch("http://localhost:8000/api/audit", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ code: "your_code_here" }),
});

const result = await response.json();
console.log(`Status: ${result.final_status}`);
```

## Troubleshooting

| Problem                         | Solution                                               |
| ------------------------------- | ------------------------------------------------------ |
| `NVIDIA_API_KEY not found`      | Copy `.env.example` to `.env` and add your key         |
| `ModuleNotFoundError`           | Activate venv: `venv\Scripts\activate` (Windows)       |
| Can't connect to localhost:8000 | Check server is running; try `APP_PORT=8080` in `.env` |
| 405 Method Not Allowed          | Use `http://localhost:8000` not port 5500              |
| Rate limit exceeded             | Wait a few minutes; iterations are capped at 3         |
